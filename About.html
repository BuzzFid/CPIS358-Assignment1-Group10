About Us

FCIT Event Finder is a student project made for the CPIS358 (Internet Applications and Web Programming course.) The goal of this project is to help FCIT students find all faculty events in one place instead of searching through WhatsApp groups and social media posts.

We built this website to make things easier. It collects events, shows clear details, and helps students know what is happening inside the faculty. Whether it's workshops, club activities, or competitions, the website makes it simple to find and follow events.

This project is created by FCIT students, for FCIT students easy to use, organized, and useful.
